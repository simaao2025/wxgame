# 分包pck加载文件系统优化注意:
- 默认加载pck会保持在内存, 当你有多个pck额外加载的时候注意内存控制及时释放, 下面是是否pck在内存中的占用, 释放不影响功能代码逻辑,只是没有在内存的文件会经历一个加载过程, 自行进行合理分割安排
```gdscript 从内存卸载pck
	var sdk = JavaScriptBridge.get_interface("GODOTSDK")
	var result = sdk.releasePck(current_pck)
	if result:
		print("pck卸载成功")
```
